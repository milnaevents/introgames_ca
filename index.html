Word Connections

Game Setup

  • Player 1 presses Start button and announces first word
  • Rotate clockwise, each person says one word at a time
Rules
  1. Word must be related to previous word (and can be unrelated to first word)
  2. Cannot repeat word during same sequence
  3. Max 5 seconds per turn
If any rule is broken, start again by pressing Next button.

Example
Balloon -> Red -> Stop -> Sign -> Rubber (Rubber is not related to Sign)

No English

Game Setup

  • Pick a language other than English
  • Start the timer
  • Start a conversation, perhaps asking questions to one another
Rules
  1. Max 5 seconds to think
  2. First person to use an English word is out
  3. Exception: Other person must know correct translation. If translation not known, game continues. Example: Computer may not have a translation.
Continue the contest until someone breaks a rule!
